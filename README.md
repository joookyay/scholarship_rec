# 장학금 추천 알고리즘

## 🎮프로젝트 소개
### 연구목적
대학원생들의 페인포인트 중 하나인 장학금 정보의 분산화를 해결하고자 장학금 추천 서비스 개발을 위한 장학금 데이터 수집 및 알고리즘 구현

### 연구문제
1. 장학금들이 어떤 요소들을 기반으로 유사도를 이루고 있는지 파악
2. 시기별, 학과별, 지도교수별 장학금 수혜 분포 파악
3. 유사도 기반의 알고리즘 제안

📌 수집된 데이터 및 연구 결과는 모두 학교에 소속되어 데이터 제공 및 연구결과 공개 불가
